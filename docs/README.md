# Documentation

Ce dossier accueille les guides techniques et notes d'exploitation. La branche `dev` démarre volontairement avec une base minimale :

- `frontend/` contient l'interface Vite + Tailwind.
- `backend/` expose l'API FastAPI optionnelle.
- `docker/` regroupe l'orchestration (Caddyfile, compose).

Ajoutez ici les spécifications complémentaires au fil des itérations (schémas d'infra, décisions d'architecture, procédures d'exploitation, etc.).
